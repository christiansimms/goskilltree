Reference for p5.js

Draw shapes:

  • ellipse(x, y, w, h)
  • line(x1, y1, x2, y2)
  • rect(x, y, w, h)

Change colors:

  • background(color)
  • fill(color)
  • noFill()
  • stroke(color)
  • noStroke()

Events:

  • mouseX
  • mouseY
  • mouseIsPressed

Environment:

  • width
  • height
  • frameRate(fps)
  • save()
To see the full set of docs, go to the p5.js website.
